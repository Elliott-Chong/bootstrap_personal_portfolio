My favourite modules

FED gives us the knowledge and skills to develop effective front-end web applications using Hypertext Markup Language (HTML) and Cascading Stylesheets (CSS). We learn to use front-end web development frameworks to further enhance our ability for rapid prototyping of responsive web application.

This module provides us with an understanding of computer networking concepts and hands-on sessions on with operating systems using Command-Line Interfaces. We are taught the use of various UNIX commands / system tools for user management, software installation, network administration and configuration of services. These topics are essential and prerequisite to an Application Developer for building and deployment of a software system.
